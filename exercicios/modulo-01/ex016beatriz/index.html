Quando eu quiser
quebrar uma linha sem colocar outra tag no código, preciso adicionar a tag br.

Para adcionar símbolos como ™ é necessário colocar o logograma "&" antes da escrita e o sinal ";" depois, ambos sem espaço. Já para emoji, é necessário colocar "&#x" antes do código disponível no site emojipedia . 😉

Não devemos esquecer dos direitos autorais de imagens, vídeos e áudios. É necessário buscar nos lugares corretos como: essa imagem foi retirada do google imagens e eu escolhi a opção de com direito criativo. já essa foto é do site pexels, também foi alterado o tamanho e resolução pelo Gimp.

A tag mark é um marca texto.Já a small é para letras miúdas.Também tem a del para palavras deletadas. ins para texto inserido. Como tantas outras, exemplo: sup- sobrescrito, sub- subscrito, code- código fonte, pre- formatação, q- citação simples, blockquote- citação site, abdr- abreviação.

Podemos fazer listas não ordenadas com a tag ul e para ordenadas com ol. Veja:

  • Oi
  • tudo
  • bem
  • ?
  1. Sim
  2. e
  3. com
  4. você
  5. ?

É possivel mudar a ordem dos números, trocar simbolo e fazer oura listagem dendo de uma lista.Basta usar as tags type, start.

Para links externos usamos a tag a e para não abrir na mesma página usamos target _blank.

Já links internos, usamos página 2 para abrir em outra aba ou página 2 para abrir na mesma.
